## 🎯 Konsept: "Neon-Cyber Visual Identity"

> Festivalin dili. Gözlerin TEKNOFEST'i ilk gördüğü an.

**Tema:** Dijital Dönüşüm → Geometrik akışlar · Neon siyan · Derin uzay siyahı

---

## 📁 Klasör Yapısı

```
grafik_konsept/
├── logo/            → Logo ana dosyaları (.ai, .svg, .png — tüm varyantlar)
├── brand_book/      → Marka rehberi PDF ve kaynak dosyaları
├── ic_mekan/        → İç mekân uygulamaları (tabela, yönlendirme, stand)
├── dis_mekan/       → Dış mekân uygulamaları (banner, bayrak, yol işareti)
├── dijital/         → Ekran görselleri ve animasyon şablonları
├── baski/           → Basılı materyal şablonları (broşür, katalog)
├── sosyal_medya/    → Sosyal medya şablonları (Instagram, Twitter, LinkedIn)
└── STYLE_GUIDE.md   → Renk, tipografi, grid, kullanım kuralları
```
